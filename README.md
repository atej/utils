# Utilities

## Installation

```shell
# npm
npx jsr add @atej/utils

# deno
deno add jsr:@atej/utils
```

## Usage

Import a utility:

```typescript
import { parseNumber } from '@atej/utils'
```

Import a type utility:

```typescript
import type { Equal } from '@atej/utils'
```
